Authors can be added as a list of their github URLs

```mdx
---
title: Authors!
date: November 26, 2023
authors:
    - https://github.com/versecafe
    - https://github.com/vercel
    - https://github.com/example
release: "0.0.4"
---
```

## Backwards Compatibility

Older logs without authors will still function fine and no author element will be rendered,
because of this version **0.0.4** is fully backwards compatible with all prior versions
despite bringing in the authors field.
